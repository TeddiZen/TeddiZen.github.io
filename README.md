# 舞萌人格测试

一个基于 Vue 2 的舞萌（MaiMai）风格人格测试网页应用。
通过回答一系列问题，测试用户的人格类型与舞萌中的哪种 Note 最相似。

## 测试流程

1. **开始页面**：输入昵称，点击开始测试
2. **答题页面**：完成 15 道音游相关情景题
3. **结果页面**：展示匹配的 Note 人格、符合度、详细描述

---

## 维度说明

测试基于四个维度计算：

- **勇敢指数**：冒险精神、果断程度、行动力
- **力量指数**：耐力、坚持性、稳定性
- **敏锐指数**：专注度、追求完美程度
- **轻松指数**：心态松弛度、包容与随和程度

测试使用加权特征线性匹配算法，根据用户回答的每个问题，计算出与每个 Note 的符合度，得到符合度最高的 Note 作为测试结果。
